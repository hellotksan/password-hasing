# Password Hashing Project

パスワードをハッシュ化するだけのプログラムファイルです。

## 使い方

0. プロジェクトのクローン

   ```bash
   git clone https://github.com/hellotksan/password-hasing
   ```

1. モジュールのインストール

   ```bash
   npm i
   ```

2. プログラム実行（ハッシュ化）

   index.js

   ```javascript
   hashPassword("hashing-test");
   ```

   この関数の引数にハッシュ化したい文字列を入力し、実行すれば
   ハッシュ化された文字列がコンソール上に出力されます。
   ただそれだけ。
